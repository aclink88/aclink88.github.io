NLP Music Evolution

Natural Language Processing project analyzing the evolution of sentiment & topic modeling across the most popular songs of the past 50 years using the Billboard Hot 100 year end rankings to represent popularity.

(GitHub repo link)

Classification Methods: Disaster Relief

Testing of different classification methods in relation to the 2010 Haitian earthquake relief effort. This project involved training various models on imagery data collected during the relief efforts to try and correctly locate displaced persons, who were known to be using blue tarps as temporary shelters. The main purpose was to compare performance metrics between the various classification algorithms, such as accuracy (using cross-validation and hold out data) and runtime.

(GitHub repo link)

Music Feature Analysis by Decade

Analysis exploring which models can accurately predict the decade in which a song was produced using audio features, both categorical and numerical data, provided via Spotify. The models were trained using PySpark (Python API for Apache Spark) to utilize its distributed computing framework for large-scale data processing.

(GitHub repo link)

Bayesian Economic Analysis

A Bayesian analysis of how certain economic indicators, such as industry employment distribution and educational attainment affect the median income of individual census tracts within subdivisions of Miami-Dade County. This analysis was conducted utilizing a hierarchical regression, in which observations in the data fall into nested levels.

(GitHub repo link)

CNN Driver Drowsiness Detection

Driver drowsiness detection using deep learning methods, specifically various convolutional neural networks. This detection system uses Python, Open CV and CV Lib to detect drowsiness using facial landmarks of driver images. CNN architectures such as ResNet 50, Inception V3 and a custom-built CNN were implemented to evaluate the drowsiness system.

(GitHub repo link)

Miami COVID Economic Recovery

Expansion on the Economic Analysis project, where further modeling was done to answer additional questions, like what the main drivers of potential economic growth are, as well as how the composition of industry employment was affected by the COVID-19 pandemic. PanelOLS and Dirichlet regressions were utilized to help answer these questions.
